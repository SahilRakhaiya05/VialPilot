Getting Started

Install VialPilot, connect Cerebras, and run your first autonomous lab workflow in under five minutes.

Prerequisites

  • Python 3.9 or newer
  • Cerebras API key for live Gemma 4 demo
  • Git (optional)

Install

git clone https://github.com/SahilRakhaiya05/VialPilot.git
cd VialPilot
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate

pip install -r requirements.txt
cp .env.example .env

Configure

Edit .env and add your Cerebras key:

CEREBRAS_API_KEY=your_key_here
CEREBRAS_MODEL=auto
PORT=7860

Without a key, the app runs in offline mock mode (tests only — not for live demo).

Run

python app.py
URLPage

http://127.0.0.1:7860/settingsAI config + benchmark

60-second demo

  1. Open Dashboard → select Hazard Avoidance scene
  2. Upload a PNG/JPG or MP4 video (or Simulator Capture)
  3. Instruction: "Move the red sample vial to the safe tray and avoid the contaminated zone."
  4. Click ▶ Start Workflow → watch agent stepper + Speed in Action panel
  5. Open Simulator → arm sweep animation
  6. Click ⚡ Speed Benchmark → show live Gemma 4 latency

Deploy docs on Vercel

This documentation site deploys from the website/ folder:

# Install Vercel CLI
npm i -g vercel

# From repo root
vercel

Or connect your GitHub repo in the Vercel dashboard — vercel.json is already configured.
